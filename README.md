# web-guvenligi-temelleri
Web sitelerinde sık karşılaşılan güvenlik risklerini ve temel koruma yöntemlerini açıklayan, giriş seviyesinde hazırlanmış bilgilendirici bir kaynak.
## ❓ Bu Repository Neden Oluşturuldu?

Bu repository, web güvenliği konusunda yeni başlayanlar için
temel farkındalık oluşturmak amacıyla hazırlanmıştır.

Günümüzde birçok web sitesi, basit ama kritik güvenlik önlemleri
alınmadığı için saldırılara açık hale gelmektedir.
Bu çalışma; teknik detaya boğulmadan, web güvenliğinin temel
kavramlarını anlaşılır bir şekilde açıklamayı hedefler.

Amaç; özellikle öğrenciler, stajyerler ve kendi web sitesini yöneten
kişiler için yol gösterici bir kaynak sunmaktır.
